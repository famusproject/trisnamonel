## Cara Menggunakan Gambar

### Opsi 1: Menggunakan Gambar yang Sudah Di-generate
Gambar-gambar di atas sudah tersedia di artifacts. Anda bisa:
1. Download gambar dari panel artifacts
2. Simpan ke folder `images/`
3. Update HTML untuk menambahkan tag `<img>`

### Opsi 2: Update HTML dengan Gambar
Tambahkan gambar ke product cards di `index.html`:

```html
<!-- Contoh untuk Gelang Premium -->
<div class="product-card" data-aos="fade-up" data-aos-delay="100">
    <div class="product-image" style="background-image: url('images/bracelet_product.png');">
        <div class="product-badge">Bestseller</div>
        <div class="product-overlay">
            <button class="view-details">Lihat Detail</button>
        </div>
    </div>
    <!-- ... rest of the card -->
</div>
```

### Opsi 3: Update CSS untuk Background Image
Tambahkan di `style.css`:

```css
.product-card:nth-child(1) .product-image {
    background-image: url('images/bracelet_product.png');
    background-size: cover;
    background-position: center;
}

.product-card:nth-child(2) .product-image {
    background-image: url('images/necklace_product.png');
```

## Tips Fotografi Produk

Jika Anda ingin menggunakan foto produk asli:

### Pencahayaan
- Gunakan natural light atau softbox
- Hindari shadow yang terlalu keras
- Pastikan warna akurat

### Background
- Gunakan background putih, beige, atau cream
- Konsisten dengan palet warna website
- Bersih dan minimalis

### Komposisi
- Foto dari berbagai angle
- Close-up untuk detail
- Lifestyle shots untuk context

### Editing
- Adjust brightness dan contrast
- Color correction untuk konsistensi
- Crop dengan rasio yang sama (square atau 4:3)

## Ukuran Gambar yang Direkomendasikan

- **Product Cards**: 800x800px (square)
- **About Section**: 1200x800px (landscape)
- **Hero Background**: 1920x1080px (full HD)
- **Logo**: 500x500px (transparent PNG)

## Format File
- **Preferred**: PNG (untuk transparansi)
- **Alternative**: JPG (untuk file size lebih kecil)
- **Optimize**: Gunakan tools seperti TinyPNG untuk kompresi
